### 分類問題に「最小二乗法」が合わない理由

[nanj:10] 彡(ﾟ)(ﾟ)「ここまでは『予測の仕組み』や。問題は『どうやってこの直線を学習するか』やな。回帰の時と何が違うか考えてみよう」

[nanj:11] 彡(ﾟ)(ﾟ)「第2話で回帰をやった時は、予測値と実測値の差を二乗してLoss（損失）にする『最小二乗法』を使った。分類でもこれでええと思うか？」

[yaruo:12] （ ＾ω＾）「えーと……実測値（正解）はラベルの『0』か『1』だお。でも予測値の `w1*x1 + w2*x2 + b` って、1万とかマイナス1万とか、無限の範囲になっちゃうお？ それと『0・1』を直接引き算するのは、なんか違和感があるお……」

[nanj:13] 彡(^)(^)「完璧なツッコミや！ そう、分類では予測値をそのまま使うんやなくて『確率』として扱いたいんや。『グループBである確率は 0.8（80%）やで』みたいに、値を『0〜1』の範囲に押し込みたい」

[nanj:14] 彡(ﾟ)(ﾟ)「ここで！ 第4.5話でチラッと紹介したあの関数が伏線回収で登場するで。どんな入力でもなめらかに0〜1の範囲に押し込む、S字カーブのアイツや！」

> [図3：sigmoid関数のグラフ]

[yaruo:15] （ ＾ω＾）「あっ！ 『sigmoid（シグモイド）』だお！！ 『二値分類の出力層で今でも現役』って言ってたのはこのことだったんだお！」

[nanj:16] 彡(^)(^)「せや！ だから予測の式はこうなる。`p = σ(w1*x1 + w2*x2 + b)`。この `p` がズバリ『グループBである確率』を表すんや」

[nanj:17] 彡(ﾟ)(ﾟ)「確率 `p` が 0.5 より大きければB、小さければAと判定する。境界線である `p = 0.5` になるのは、カッコの中身がちょうど `0` になる時……つまり直線 `w1*x1 + w2*x2 + b = 0` の上ってわけや」

[yaruo:18] （ ＾ω＾）「なるほどお！ sigmoidを通しても『直線の境界線』という事実は変わらないけど、出てくる数字が『確率』として解釈できるようになるんだお！」

### 知ったかぶりを絶対に許さない「交差エントロピー」

[nanj:19] 彡(ﾟ)(ﾟ)「確率が出せるようになったら、次はLoss（損失）の定義や。分類では最小二乗法ではなく、『交差エントロピー誤差（Cross Entropy Error）』っていう中二病みたいな名前のLossを使うで」

[nanj:20] 彡(ﾟ)(ﾟ)「1つのデータ点に対する交差エントロピーは、ラベル `y`（0 か 1）と予測確率 `p` を使って次のように書く」
`L = -{ y * log(p) + (1 - y) * log(1 - p) }`

[yaruo:21] （；＾ω＾）「うぎゃあああ！ 突然 `log` が出てきて頭が爆発したお！ なんでこんなゴチャゴチャした呪文みたいな式を使うんだお！」

[nanj:22] 彡(-)(-)「落ち着け。数式アレルギー発症すな。直感的に理解すれば超シンプルや。例えば、正解ラベルが `y = 1`（B）の場合、後半が消えて式は `L = -log(p)` になる」

[nanj:23] 彡(ﾟ)(ﾟ)「予測確率 `p` が 1（100%）に近ければ、正解してるから `log(p)` は 0 に近づいて Loss は小さくなる。逆に `p` が 0（0%）に近いと、グラフを見てみい。`log(p)` はマイナス無限大に発散して、Lossが宇宙規模に大爆発するんや！」

> [図4：-log(p) のグラフ（p=1でLossは0だが、p=0に近づくとLossが無限大に発散して跳ね上がる）]

[yaruo:24] （ ＾ω＾）「……！ つまり『正解はBなのに、AIが「絶対にAです！（Bである確率は0%）」って自信満々に間違えた』時に、ペナルティが無限大に大きくなるってことかお！？」

[nanj:25] 彡(^)(^)「その通りや！ ネットのレスバでも『いや、絶対に〇〇だしｗｗ』って知ったかぶりしてドヤ顔で間違えた奴が、一番ボコボコに叩かれてダメージ（Loss）でかいやろ？ それと同じや」

[nanj:26] 彡(ﾟ)(ﾟ)「最小二乗法やと、どれだけ派手に間違えても Loss は高々 `(1-p)^2`（最大1）で頭打ちになってまう。でも交差エントロピーは『自信を持って間違えた予測』を無限に強く罰する。これが分類問題の学習をガンガン進める秘訣なんやで」

[yaruo:27] （ ＾ω＾）「知ったかぶり無能AIを絶対に許さない、スパルタ教育の数式だったんだお！」

### まとめ：分類で変わるもの、変わらないもの
